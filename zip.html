Si të shkarkojë aplikacionin ?

Keto janë Instrukcionet !

1. Një version i Python 3.10.10 ose më i vjetër (3.11 nuk funksionon)

2. Packages apo Librarit e python:

  1. - Mediapipe
  2. - Opencv-python
  3. - Numpy
  4. - Tkinter
  5. - Pillow

3. Ju mund te instaloni keto Librari duke perdorur:

  1. - pip install (package), për shembull: pip install mediapipe
  2. - python -m pip install (package), për shembull: python -m pip install opencv-python
  3. - py -m pip install (package), për shembull: py -m pip install tkinter

    4. 4. Perdorimi i komandave

    Këto komanda duhet të shkruhen në Powershell apo në CMD (Command Prompt). Të gjithë këto mënyra punojnë njëjtë, por varen nga ju se cila ju punon. Ju mund të përdorni secilën prej këtyre që keni dëshirë.

    5. ZIP file

      - Pasiguria se keni të gjitha gjërat e nevojshme për të provuar këtë program, thjesht bëni extract zip që ju downloaduat dhe hapni aplikacionin.

      - Te bashkangjitur keni dy linqet ku mund te shkarkoni Python - 3.10 dhe aplikacionin ne formen e ZIP file.
